# dotfiles

Dotfiles, inc files for:

* ack
* beets
* csi
* git
* jupyter (custom CSS)
* matplotlibrc
* mpd
* R
* todo.txt
* tmux
* vim
* weechat
* xinitrc

## Installation

Run `make`.  This creates many symlinks after first creating backups of the symlink targets.
